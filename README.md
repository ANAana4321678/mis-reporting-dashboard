# 📊 MIS Reporting & Sales Performance Dashboard

## Project Overview
This project demonstrates an automated MIS reporting solution designed to track sales performance and generate business insights.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Key Features
- Sales trend analysis
- Region-wise performance
- Target vs achievement
- Automated reporting

## Business Impact
Reduced manual reporting effort and improved decision-making efficiency.
